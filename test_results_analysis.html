If x² - 5x + 6 = 0, then the values of x are:

A x = 1, 6 Your Answer
B x = 2, 3 Correct Answer
C x = -2, -3
D x = 5, 1
Explanation:

To solve x² - 5x + 6 = 0, we can factorize the quadratic equation:

x² - 5x + 6 = (x - 2)(x - 3) = 0

Therefore, x = 2 or x = 3
